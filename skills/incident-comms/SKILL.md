# Incident Communications

Generate stakeholder communications from investigation findings.

## When to Use

- During active incidents — status page updates
- After resolution — exec summary for leadership
- Post-incident — postmortem draft

## Output Types

- **Status Update**: Brief, factual, suitable for status page
- **Exec Summary**: High-level impact and resolution for leadership
- **Postmortem Draft**: Full structured postmortem document
