# RCA Template

Structured root cause analysis generation.

## When to Use

- After an incident is resolved
- When preparing a postmortem document
- To generate a structured timeline of events

## Output

- Incident timeline with key events
- Root cause identification
- Blast radius analysis
- Corrective actions (immediate and long-term)
