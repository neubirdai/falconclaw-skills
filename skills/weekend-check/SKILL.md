# Weekend Check

Risk assessment for oncall handoffs.

## When to Use

- Before starting an oncall shift
- Friday afternoon pre-weekend review
- Any handoff between oncall rotations

## What It Checks

- Recent deployments and config changes
- Capacity trends approaching thresholds
- Services with elevated error rates
- Known fragile spots and upcoming maintenance windows
